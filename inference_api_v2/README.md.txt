# 🌾 Agro Data Fetcher

This mini toolkit fetches **NDVI**, **average rainfall**, and **temperature** data for any farm location using public APIs.

## 📁 Files
- `get_ndvi_average.py` → Fetches NDVI average for the past 6 months using AgroMonitoring API.
- `get_rainfall_temp.py` → Fetches average rainfall and temperature for the past 6 months using Open-Meteo API.
